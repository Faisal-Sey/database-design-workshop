---

## Talking points

- Welcome. Session is 2.5 hours with two hands-on breaks.
- Everything is in the repo — no need to copy code frantically.
- Three things you'll leave with:
  1. A framework for picking the right database
  2. The ability to read a slow query and fix it
  3. A list of mistakes to avoid before they cost you

## Ice-breaker

> "Name one Ghanaian app you use every day that you think has an interesting database problem."

Listen for: MoMo, GhanaPostGPS, student portals, ride-hailing, betting apps.
Validate every answer briefly — they're all real database problems.
